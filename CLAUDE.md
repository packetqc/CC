# Learn Coding Skills

A structured, hands-on project for learning essential coding skills through practice.

## Structure

- `skills/` - Skill modules organized by topic
- `exercises/` - Practical coding exercises with tests
- `solutions/` - Reference solutions

## Running Exercises

```bash
# Python exercises
python -m pytest exercises/ -v

# JavaScript exercises
node exercises/js/run.js
```

## Skill Tracks

1. **Data Structures** - Arrays, linked lists, trees, graphs, hash maps
2. **Algorithms** - Sorting, searching, dynamic programming, recursion
3. **Design Patterns** - Creational, structural, behavioral patterns
4. **Problem Solving** - Breaking down problems, edge cases, optimization
